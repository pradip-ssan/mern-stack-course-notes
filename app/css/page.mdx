# CSS

To become an **advanced CSS developer**, you need to go beyond the basics and master modern CSS techniques, performance optimizations, and best practices.

## Step 1: Master the Fundamentals

Before diving into advanced CSS, ensure you have a strong grasp of the basics.

### ✅ Selectors & Specificity

- Universal, Type, Class, ID, Attribute selectors
- Advanced selectors (:not(), :nth-child(), :nth-of-type(), ::before, ::after)
- Specificity rules & !important

### ✅ Box Model

- Content, Padding, Border, Margin
- box-sizing: border-box;

### ✅ Positioning & Display

- static, relative, absolute, fixed, sticky
- inline, block, inline-block, flex, grid

### ✅ Typography & Colors

- rem, em, vh, vw, %
- clamp(), calc(), var()

### ✅ Flexbox & Grid Basics

- display: flex; properties (justify-content, align-items, flex-wrap, etc.)
- display: grid; properties (grid-template-columns, grid-template-rows, gap, etc.)

### ✅ Transitions & Animations

- transition: all 0.3s ease;
- @keyframes and animation

## Step 2: Responsive Design & Layout Mastery

At an advanced level, you should be able to build fully responsive layouts without using frameworks like Bootstrap / Tailwind css.

### ✅ Advanced Flexbox & Grid Layouts

- Nesting flex and grid elements
- grid-auto-flow, grid-template-areas
- Creating full-page layouts

### ✅ CSS Media Queries

- Mobile-first vs. Desktop-first approach
- @media (max-width: 768px) {}

### ✅ Container Queries (New in CSS)

- @container for responsive components

### ✅ Fluid & Adaptive Typography

- clamp() for scalable fonts
- calc() for dynamic spacing

### ✅ CSS Viewport Units

- vh, vw, vmin, vmax, svh, lvh, dvh

### ✅ CSS Aspect Ratio

- aspect-ratio: 16 / 9;

## Step 3: CSS Custom Properties (Variables)

### ✅ Using CSS Variables

example:-

```
:root {
  --primary-color: #007bff;
  --font-size: 16px;
}

body {
  color: var(--primary-color);
  font-size: var(--font-size);
}
```

### ✅ Dynamic Themes with CSS Variables

- Dark mode implementation
- Theming with CSS variables

## Step 4: Advanced CSS Animations & Effects

### ✅ CSS Keyframe Animations

- Smooth transitions using @keyframes
- CSS-only loaders and spinners

### ✅ Scroll-Based Animations

- scroll-behavior: smooth;
- @scroll-timeline

### ✅ Parallax Effects using CSS

- perspective and transform

### ✅ 3D Effects & Transformations

- transform: rotateX(45deg);
- perspective: 1000px;

### ✅ Glassmorphism, Neumorphism, & Frosted Glass

Example:-

```
.glass {
  background: rgba(255, 255, 255, 0.2);
  backdrop-filter: blur(10px);
}
```

### ✅ CSS Clip-Path & Masking

- clip-path: polygon();
- mask-image: linear-gradient();

### ✅ GSAP for Advanced Animations

- JavaScript-powered animations with CSS

## Step 5: CSS Performance Optimization

### ✅ Minimizing CSS File Size

- Remove unused CSS
- Use minified CSS

### ✅ Critical CSS & Lazy Loading

```
- Load essential CSS first (<link rel="preload" as="style">)
```

### ✅ Avoiding Render Blocking

- defer CSS when needed

### ✅ Hardware-Accelerated Animations

- Use transform: translate3d(0, 0, 0);

### ✅ Reducing Repaints & Reflows

- Optimize CSS selectors
- Use will-change only when necessary

### ✅ Use of Subgrid (Latest in CSS Grid)

- display: subgrid;

- class="flex justify-center items-center"

### ✅ Extending Tailwind with Custom Styles

- @apply, extend, theme

### ✅ Optimizing Tailwind CSS for Production
