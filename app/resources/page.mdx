# Recources

## best learning and web development resources.

### 🛠️ 1. Documentation & Official References

- MDN Web Docs → https://developer.mozilla.org

- JavaScript.info → https://javascript.info

- React Docs → https://react.dev

- Next.js Documentation → https://nextjs.org/docs

### 🧰 2. Tools to Learn

- Visual Studio Code (editor) → VS Code Docs

- Git + GitHub (version control) → GitHub Docs

- Tailwind CSS (styling) → Tailwind Docs

- Vercel (hosting) → Vercel Docs

- Netlify (hosting) → Netlify Docs

- Postman (API testing) → Postman Learning Center

### ⭐ Pro Tips

- Build projects early (even small ones: calculator, todo app, portfolio site).

- Learn Git/GitHub from the beginning.

- Write code every day (even 30 minutes daily compounds fast).

- Read docs and official sites — don't rely only on tutorials.

- Host your projects — even basic ones — on Vercel, Netlify, or GitHub Pages.
